# collection集合

java中集合框架的顶层是 collection集合 和 map集合
collection集合中操作元素的基本单位是：单个元素
map集合中操作元素的基本单位是：单对元素

collection:List、queue、set
List  ->  ArrayList、LinkedList、Stack、Vector
queue
set  ->  HashSet、TreeSet

def:
        添加：add
        删除：remove
        长度：size
        清空：clear
        判断包含：contains
        判断为空：isEmpty
