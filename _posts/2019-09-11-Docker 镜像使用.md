# Docker 镜像使用

## 列出镜像列表
```
docker images
```

如果要使用版本为18.04的ubuntu系统镜像来运行容器时，命令如下：
```
docker run -t -i ubuntu:18.04 /bin/bash
```

## 查找镜像
```
docker search httpd
```

## 拖取并使用镜像
```
docker pull httpd
docker run httpd
```

## 更新镜像
```
使用镜像创建容器：docker run -t -i ubuntu:15.10 /bin/bash
容器内更新：apt-get update
退出容器：exit
```
